---
title: "Facebook Page Info"
api: "GET https://piloterr.com/api/v2/facebook/page/info"
description: "Get Facebook page info"
---

### Header

<ParamField header="x-api-key" type="string" required={true}>
    Parameter defines the Piloterr private key to use.
</ParamField>

### Parameters

<ParamField query="query" type="string" optional={false} placeholder="actutoulouse">
    Facebook page url or slug : `https://www.facebook.com/actutoulouse` or `actutoulouse`
</ParamField>

<RequestExample>

```bash Actu Toulouse
curl --location --request GET 'https://piloterr.com/api/v2/facebook/page/info?query=actutoulouse' \
--header 'Content-Type: application/json' \
--header 'x-api-key: <token>' \
```

