---
title: "Crunchbase Funding Rounds"
api: "GET https://piloterr.com/api/v2/crunchbase/funding_rounds"
description: "Get funding rounds of companies from Crunchbase"
---

### Header

<ParamField header="x-api-key" type="string" required={true}>
    Parameter defines the Piloterr private key to use.
</ParamField>

### Parameters

<ParamField query="days_since_announcement" type="integer" placeholder="30">
    Parameter defines the number of days since the announcement of the funding round : `30`, `60`, `90`, `360`. Default value is `30`.
</ParamField>

<ParamField query="investment_type" type="string" placeholder="seed">
    Parameter defines the type of the investment : `pre_seed`, `seed`, `series_a`, `series_b`, `series_c`, `series_d`, `series_e`, `series_f`, `series_g`, `series_h`, `series_i`, `series_j`, `series_unknown`, `angel`, `private_equity`, `debt_financing`, `convertible_note`, `grant`, `corporate_round`, `equity_crowdfunding`, `product_crowdfunding`, `secondary_market`, `post_ipo_equity`, `post_ipo_debt`, `post_ipo_secondary`, `non_equity_assistance`, `initial_coin_offering`, `undisclosed`.
</ParamField>

<ParamField query="investor_identifiers" type="string" placeholder="df14597c-3e51-8d58-9c15-fb3de3a34a54">
    Parameter defines the UUID of the investor.
</ParamField>
