---
title: "Crunchbase Company Info"
api: "GET https://piloterr.com/api/v2/crunchbase/company/info"
description: "Get company information from Crunchbase"
---

<Tip>
    It is possible to get information about the company from Crunchbase using a URL linked to the company (crunchbase, linkedin, twitter or facebook).
</Tip>

### Header

<ParamField header="x-api-key" type="string" required={true}>
    Parameter defines the Piloterr private key to use.
</ParamField>

### Parameters

<ParamField query="query" type="string" optional={false} placeholder="https://www.crunchbase.com/organization/openai">
    A query to search for a company. It can be: **Crunchbase URL**, **Website URL**, **Linkedin URL**, **Facebook URL** or **Twitter URL**.
</ParamField>

<RequestExample>

```bash openai
curl --location --request GET 'https://piloterr.com/api/v2/crunchbase/company/info?query=https://www.crunchbase.com/organization/openai' \
--header 'Content-Type: application/json' \
--header 'x-api-key: <token>' \
```

</RequestExample>
