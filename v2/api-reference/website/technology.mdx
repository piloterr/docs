---
title: "Technology"
api: "GET https://piloterr.com/api/v2/website/technology"
description: "Get website technology (CMS, Framework, Analytics, CDN, Hosting, etc.)"
---

<Warning>
    Response time is 10-15 seconds.
</Warning>

### Header

<ParamField header="x-api-key" type="string" required={true}>
    Parameter defines the Piloterr private key to use.
</ParamField>

### Parameters

<ParamField query="query" type="string" optional={false} placeholder="https://example.com">
    A website url with http(s) protocol
</ParamField>

### Response

<RequestExample>

```bash leroymerlin.fr
curl --location --request GET 'https://piloterr.com/api/v2/website/technology?query=https://www.leroymerlin.fr' \
--header 'Content-Type: application/json' \
--header 'x-api-key: <token>' \
```

</RequestExample>
