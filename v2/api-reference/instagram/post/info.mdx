### Header

<ParamField header="x-api-key" type="string" required={true}>
    Parameter defines the Piloterr private key to use.
</ParamField>

### Parameters

<ParamField query="query" type="string" optional={false} placeholder="https://www.instagram.com/p/CshTQQlMGMl">
    Instagram post or reel url : `https://www.instagram.com/p/CshTQQlMGMl` or `CshTQQlMGMl`
</ParamField>
