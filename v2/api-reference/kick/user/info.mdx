---
title: "Kick User Info"
api: "GET https://piloterr.com/api/v2/kick/user/info"
description: "Get Kick user info"
---

### Header

<ParamField header="x-api-key" type="string" required={true}>
    Parameter defines the Piloterr private key to use.
</ParamField>

### Parameters

<ParamField query="query" type="string" optional={false} placeholder="https://kick.com/xqc">
    Kick user url : `https://kick.com/xqc` or `xqc`
</ParamField>

</ResponseField>

```bash xqc user info
curl --location --request GET 'https://piloterr.com/api/v2/kick/user/info?query=xqc' \
--header 'Content-Type: application/json' \
--header 'x-api-key: <token>' \
```
