<Accordion title="How do I signup for the API?">

    The Radiant Segment API is currently in private beta and we're inviting new developers in batches. If you want to get early access and try out the API, please [sign up](https://piloterr.com/register).

</Accordion>

<Accordion title="Is the API free to use?">

    Yes, the API can be explored in a free plan that comes with certain limitations, to use the full capabilities a paid plan can be purchased.

</Accordion>

<Accordion title="I have some questions, where can I get help? ">

    Please reach out on our [Discord Server](https://discord.gg/Y7U5Ka73yT), we're happy to answer you all your questions and get you started.

</Accordion>

<Accordion title="How requests are counted?">

    Only successful requests are counted towards your monthly searches. Cached, errored and failed searches are not.

</Accordion>

<Accordion title="Can I request that Piloterr adds an API for scraping?">

    Of course! We are always expanding our code and building new APIs for different search engines. You can create a feature request on our Roadmap. We aren't able to build APIs for every websites for a variety of reasons but we do always make additions where they are needed and able to be done.

    [Roadmap](https://piloterr.com/roadmap)

</Accordion>

<Accordion title="Do I keep my requests the following month?">

    No, your requests are reset at the beginning of each month. You can check your remaining credits in the dashboard.

</Accordion>
